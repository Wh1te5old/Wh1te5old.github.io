1.它的排序方法类似于冒泡动作，所以起名为冒泡排序法。另外，这也是学校考试和公司面试的常见考题。对于学习程序算法来说冒泡排序是一个很好的入门题目。
题目:创建数组,使用冒泡排序给数组从小到大排序

	int num[] = new int[]{9,6,3,5,1,2,4,7,8};	//创建数组
	System.out.print("排序前:");					//输出排序前数组
	for (int i = 0; i < num.length; i++) {
	System.out.print(num[i]+",");
	}
	
	for (int i = num.length-1; i > 0; i--) {	//十个数据比较次数应该为9次,所以i的值应该是num.length-1,每次循环比较完,最大数就不用比较了,所以i--
		for (int j = 0; j < i; j++) {
			if (num[j] > num[j+1]) {			//条件:前面的数比后面数大
			    int temp = num[j];			    //将前面的值复制给temp变量
			    num[j] = num[j+1];				//将后面的值赋值给前面的
			    num[j+1] = temp;				//将temp赋值给后面的数
			}
					
		}
	}
	
	System.out.print("\n排序后:");				//输出排序后数组
	for (int i = 0; i < num.length; i++) {
	System.out.print(num[i]+",");
	}
