Mysql8.0安装

2017-05-09 2 min read # Mysql

记录mysql8.0安装和修改密码

1.下载mysql8.0

2.新建my.ini文件并加入以下

[ mysqld ]

设置3306端口

port=3306

设置mysql的安装目录

basedir="E:\mysql-8.0.19"

设置mysql数据库的数据的存放目录

datadir="E:\mysql-8.0.19\Data"

允许最大连接数

max_connections=200

允许连接失败的次数。

max_connect_errors=10

服务端使用的字符集默认为utf8mb4

character-set-server=utf8mb4

创建新表时将使用的默认存储引擎

default-storage-engine=INNODB

默认使用“mysql_native_password”插件认证

#mysql_native_password
default_authentication_plugin=mysql_native_password
[mysql]

设置mysql客户端默认字符集

default-character-set=utf8mb4
[client]

设置mysql客户端连接服务端时默认使用的端口

port=3306
default-character-set=utf8mb4

注意:basedir和datadir里面写自己的路径

4.初始化mysql并登录

(1)用管理员身份打开cmd,然后进到mysql文件夹里的bin目录,输入mysqld --initialize --console,如果成功会出现一个初始密码.
(2)安装MySQL服务：mysqld --install
(3)登陆数据库并输入初始密码: mysql -u root -p
(4)进入数据库修改root账号密码,输入:ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY '你的密码';
(5)使用navicat远程连接mysql
