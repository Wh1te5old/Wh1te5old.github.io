Linux基本操作

cd	/	 跳到根目录下面
cd	~	跳到家的目录下面
l1		   显示当前目录下面的所有文件和文件夹
cd		  路径地址跳转到某一个路径下面
Su		 用户名切换到某一个用户 (如果不输入用户名，就是切换到管理 员账号)
pwd	  查看当前所处的路径位置
cd..	    网上跳-级
mkdir	新建文件夹
mkdir  -p	递归新建路径 下的多个文件夹	html/a/b/c/d
rmdir	删除文件夹
rmdir -p	递归删除路径 下的多个文件夹htm1/a/b/c/d
rm              删除文件夹或者文件
rm  -r		删除文件夹以及文件夹下面的所有文件
rm -rf	   强制删除文件夹以及文件夹下面的所有文件
cat 文件   	查看文件
cat>文件	  新建文件
移动文件或者文件夹: mv	文件	目标位置
复制文件或者文件夹: cp	 文件	目标位置
//文件的操作
vim / vi	编辑文件,文件名称/文件的全路径如果这个文件不存在,那么创建文件，如果存在 ，就打开这个文件
命令行---->	写入状态 i
命令行:无法修改文件里 面的内容,可以输入很多的指令
写入状态---> 	命令行状态 esc
写入状态:		 无法输入指令，可以修改文件的内容
命令行状态:
保存并且退出         : wq
不保存强制退出         :q!
显示行号         :set nu
删除一行         dd
查找          /查找的内容
跳转到最后一行         shift+g
跳转到第一行         gg
u		         撤消上一个操作
Ctrl+r			         多次撤消
整页翻页命令为: Ctrl+f	键	f的英文全拼为: forward  ; Ctrl +b   键	b的英文全拼为: backWord;
写入状态:         可以往文件里面写入数据
[0]pen Read-0nly, (E)dit anyway, (R)ecover, (D)elete it, (Q)uit, (A)bort://创空的文件touch 4. sh
创建用户: useradd 用户名
切换用户: su 用户名
删除用户(包括用户自带的文件夹):userdel  -rf 用户名
查询所有的用户:	vim /etc/passwd
