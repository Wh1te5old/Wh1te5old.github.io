Linux基本操作

cd	/	 跳到根目录下面
cd	~	跳到家的目录下面
l1		   显示当前目录下面的所有文件和文件夹
cd		  路径地址跳转到某一个路径下面
Su		 用户名切换到某一个用户 (如果不输入用户名，就是切换到管理 员账号)
pwd	  查看当前所处的路径位置
cd..	    网上跳-级
mkdir	新建文件夹
mkdir  -p	递归新建路径 下的多个文件夹	html/a/b/c/d
rmdir	删除文件夹
rmdir -p	递归删除路径 下的多个文件夹htm1/a/b/c/d
rm              删除文件夹或者文件
rm  -r		删除文件夹以及文件夹下面的所有文件
rm -rf	   强制删除文件夹以及文件夹下面的所有文件
cat 文件   	查看文件
cat>文件	  新建文件
移动文件或者文件夹: mv	文件	目标位置
复制文件或者文件夹: cp	 文件	目标位置

Ctrl+r			         多次撤消

分组+权限指令

一个用户只能有一个主组, 可以有其他的附属组
新建组: groupadd j206
查看组: vim /etc/group
删除组: groupdel j206

修改用户去哪个组:

usermod -G 组名  用户名    附属组

docker

docker 安装:

    1   查看linux内核版本
        uname -a
        升级内核版本
        yum -y update   (所有的全面升级)
        yum -y upgrade   (升级软件包)
    2   安装软件包
        yum install -y yum-utils device-mapper-persistent-data lvm2    
    3   设置yum的源为 阿里云
        yum-config-manager --add-repo http://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo
    4   安装docker
        yum install docker-ce
    5   查看docker版本    
        docker -v
    6   修改docker的镜像地址
        在此路径新建配置文件 /etc/docker/daemon.json , 并且修改其中内容为以下:
        {
            "registry-mirrors": ["https://registry.docker-cn.com", "http://hub-mirror.c.163.com","https://docker.mirrors.ustc.edu.cn"],
            "insecure-registries":["192.168.135.163:5000"]
        }
    7   启动docker
        systemctl start docker
    8   重启   
        systemctl restart docker
    9     开机自启
        systemctl enable docker

docker的基本操作

1.docker的架构:
镜像    :    类
容器    :    对象

2.镜像的操作
搜索    查询   docker search 镜像名称.  
拉取           docker pull 镜像名称.  
删除           docker rmi 镜像名称
修改镜像名称   docker tag 旧名称  新名称
查看本机上有哪些镜像    docker images
//查看私服上的镜像:http://192.168.135.163:5000/v2/_catalog
//查看私服上镜像的版本:http://192.168.135.163:5000/v2/mysql/tags/list

3.容器的操作
查看运行中的容器:  docker ps
查看所有的容器, 包含未运行的:        docker ps -a
交互式创建: docker run -ti --name mysql01 mysql /bin/bash    创建完之后直接进入到了容器里面,  如果退出容器,  容器自动关闭
后台式创建 : docker run -di --name mysql02 mysql     
创建mysql容器的时候, 一定要指定root的密码:docker run -di --name mysql03 -e MYSQL_ROOT_PASSWORD=123456 mysql
进入容器:  docker exec -it mysql03 /bin/bash
删除容器: docker rm 容器名称/id
开启容器: docker start mysql03

4.日志查看
docker logs -f 容器名称

5.文件的复制
docker cp 1.txt mysql03:/usr/local

6.创建容器并且端口映射, 外网才能访问容器
docker run -di --name mysql03 -e MYSQL_ROOT_PASSWORD=123456 -p 3306:3306 mysql

7.创建容器时进行挂载:
docker run -di --name centos -v /usr/local/jdk/:/usr/local/src centos

8.创建镜像
1.通过容器创建
docker commit -m=”镜像的描述” -a=”作者” 容器id  镜像名称:版本号

    4.在当前文件夹下创建文件 Dockerfile
